`@property`, `@setter`, `@deleter`

Let's explore Python's property decorators through TDD. We'll build a temperature class with computed properties.

Step 1: Test Temperature Property (Red Phase)

Start with a test for a temperature class with property access:

properties_test.py
def test_temperature_property(): temp = Temperature() temp.celsius = 25 assert temp.celsius == 25 assert temp.fahrenheit == 77

Run the test: pytest properties_test.py -v

Expected result: The test fails because the Temperature class doesn't exist yet.

Step 2: Implement Basic Temperature Class (Green Phase)

Write the minimal code to make the test pass:

properties.py
class Temperature: def __init__(self): self._celsius = 0 @property def celsius(self): return self._celsius @celsius.setter def celsius(self, value): self._celsius = value @property def fahrenheit(self): return self._celsius * 9/5 + 32

Run the test again: pytest properties_test.py -v

Step 3: Test Fahrenheit Setter (Red Phase)

Now add a test for setting temperature in Fahrenheit:

properties_test.py
def test_temperature_property(): temp = Temperature() temp.celsius = 25 assert temp.celsius == 25 assert temp.fahrenheit == 77 def test_fahrenheit_setter(): temp = Temperature() temp.fahrenheit = 86 assert temp.fahrenheit == 86 assert temp.celsius == 30

Run the test: pytest properties_test.py -v

Expected result: The first test passes, but the second fails because fahrenheit setter doesn't exist yet.

Step 4: Implement Fahrenheit Setter (Green Phase)

Add the fahrenheit setter implementation:

properties.py
class Temperature: def __init__(self): self._celsius = 0 @property def celsius(self): return self._celsius @celsius.setter def celsius(self, value): self._celsius = value @property def fahrenheit(self): return self._celsius * 9/5 + 32 @fahrenheit.setter def fahrenheit(self, value): self._celsius = (value - 32) * 5/9

Run the test again: pytest properties_test.py -v

Expected result: Both tests now pass! ðŸŽ‰

What We Learned

  • @property: Creating getter methods that act like attributes
  • @setter: Creating setter methods for property assignment
  • @deleter: Creating deleter methods for property cleanup
  • Computed properties: Properties that calculate values on access
