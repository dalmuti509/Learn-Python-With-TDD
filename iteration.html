Loops and Iteration in Python

Let's explore Python's iteration capabilities through TDD.

Step 1: Test Basic Iteration (Red Phase)

Start with a simple function that returns a list of numbers:

iteration_test.py
def test_count_to_three(): result = count_to_three() assert result == [1, 2, 3]

Run the test: pytest iteration_test.py -v

Expected result: The test fails because count_to_three() doesn't exist yet.

Step 2: Implement Basic Iteration (Green Phase)

Write the minimal code to make the test pass:

iteration.py
def count_to_three(): return [1, 2, 3]

Run the test again: pytest iteration_test.py -v

Expected result: The test now passes! üéâ

Step 3: Test List Comprehension (Red Phase)

Now add a test for a more complex iteration using list comprehension:

iteration_test.py
def test_count_to_three(): result = count_to_three() assert result == [1, 2, 3] def test_count_even_numbers(): result = count_even_numbers(5) assert result == [2, 4]

Run the test: pytest iteration_test.py -v

Expected result: The first test passes, but the second fails because count_even_numbers doesn't exist yet.

Step 4: Implement List Comprehension (Green Phase)

Add the implementation using list comprehension:

iteration.py
def count_to_three(): return [1, 2, 3] def count_even_numbers(max_num): return [x for x in range(1, max_num + 1) if x % 2 == 0]

Run the test again: pytest iteration_test.py -v

Expected result: Both tests now pass! üéâ
