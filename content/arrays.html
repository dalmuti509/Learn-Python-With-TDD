Working with Lists and Arrays

Let's explore Python's list operations through TDD.

Step 1: Test List Filtering (Red Phase)

Start with a function that filters positive numbers from a list:

list_operations_test.py
def test_filter_positive_numbers(): numbers = [-1, 2, -3, 4, -5] result = filter_positive(numbers) assert result == [2, 4]

Run the test: pytest list_operations_test.py -v

Expected result: The test fails because filter_positive() doesn't exist yet.

Step 2: Implement List Filtering (Green Phase)

Write the minimal code to make the test pass:

list_operations.py
def filter_positive(numbers): return [x for x in numbers if x > 0]

Run the test again: pytest list_operations_test.py -v

Step 3: Test List Summation (Red Phase)

Now add a test for summing a list of numbers:

list_operations_test.py
def test_filter_positive_numbers(): numbers = [-1, 2, -3, 4, -5] result = filter_positive(numbers) assert result == [2, 4] def test_sum_list(): numbers = [1, 2, 3, 4, 5] result = sum_list(numbers) assert result == 15

Run the test: pytest list_operations_test.py -v

Expected result: The first test passes, but the second fails because sum_list doesn't exist yet.

Step 4: Implement List Summation (Green Phase)

Add the implementation for summing a list:

list_operations.py
def filter_positive(numbers): return [x for x in numbers if x > 0] def sum_list(numbers): return sum(numbers)

Run the test again: pytest list_operations_test.py -v
