Dictionaries in Python

Let's explore Python's dictionary operations through TDD. We'll build a student management system.

Step 1: Test Dictionary Creation (Red Phase)

Start with a test for creating a student manager:

student_manager_test.py
def test_student_manager_creation(): manager = StudentManager() assert manager.students == {}

Run the test: pytest student_manager_test.py -v

Expected result: The test fails because the StudentManager class doesn't exist yet.

Step 2: Implement StudentManager Class (Green Phase)

Write the minimal code to make the test pass:

student_manager.py
class StudentManager: def __init__(self): self.students = {}

Run the test again: pytest student_manager_test.py -v

Expected result: The test now passes! ğŸ‰

Step 3: Test Adding Students (Red Phase)

student_manager_test.py
def test_student_manager_creation(): manager = StudentManager() assert manager.students == {} def test_add_student(): manager = StudentManager() manager.add_student("Alice", 85) assert manager.students["Alice"] == 85

Run the test: pytest student_manager_test.py -v

Expected result: The first test passes, but the second fails because add_student method doesn't exist yet.

Step 4: Implement Add Student Method (Green Phase)

Add the minimal implementation for adding students:

student_manager.py
class StudentManager: def __init__(self): self.students = {} def add_student(self, name, grade): self.students[name] = grade

Run the test again: pytest student_manager_test.py -v

Expected result: Both tests now pass! ğŸ‰

Step 5: Test Getting Student Grade (Red Phase)

Add a test for retrieving a student's grade:

student_manager_test.py
def test_student_manager_creation(): manager = StudentManager() assert manager.students == {} def test_add_student(): manager = StudentManager() manager.add_student("Alice", 85) assert manager.students["Alice"] == 85 def test_get_student_grade(): manager = StudentManager() manager.add_student("Alice", 85) grade = manager.get_grade("Alice") assert grade == 85

Run the test: pytest student_manager_test.py -v

Run the test again: pytest student_manager_test.py -v

Expected result: All tests now pass! ğŸ‰

Step 7: Test Safe Dictionary Access (Red Phase)

Add a test for handling non-existent students safely:

student_manager_test.py
def test_student_manager_creation(): manager = StudentManager() assert manager.students == {} def test_add_student(): manager = StudentManager() manager.add_student("Alice", 85) assert manager.students["Alice"] == 85 def test_get_student_grade(): manager = StudentManager() manager.add_student("Alice", 85) grade = manager.get_grade("Alice") assert grade == 85 def test_get_nonexistent_student(): manager = StudentManager() grade = manager.get_grade("Bob") assert grade is None

Run the test: pytest student_manager_test.py -v

Expected result: The first three tests pass, but the fourth fails because the method raises a KeyError instead of returning None.

Step 8: Implement Safe Dictionary Access (Green Phase)

Update the method to handle missing keys safely:

student_manager.py
class StudentManager: def __init__(self): self.students = {} def add_student(self, name, grade): self.students[name] = grade def get_grade(self, name): return self.students.get(name)

Run the test again: pytest student_manager_test.py -v

Expected result: All tests now pass! ğŸ‰

What We Learned

  • Dictionary creation: Using {} to create empty dictionaries
  • Dictionary access: Using dict[key] to get values
  • Safe access: Using dict.get(key) to avoid KeyError
  • Dictionary assignment: Using dict[key] = value to set values
  • TDD with dictionaries: Building dictionary operations incrementally
