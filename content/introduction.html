Welcome to Learn Python with Tests!

This course teaches Python programming through Test-Driven Development (TDD). You'll learn Python by writing tests first, then implementing the code to make those tests pass.

What You'll Learn

  • Python fundamentals through TDD
  • Advanced Python concepts
  • Best practices for writing maintainable code
  • How to think in terms of tests and interfaces

Prerequisites

This course assumes you have basic programming knowledge, but no prior Python experience is required.

Setup Instructions

Before you begin, you'll need to set up your development environment. Follow these steps to get everything ready:

1. Install Python

Download Python: Visit python.org/downloads and download the latest Python 3.x version

Important: Make sure to check "Add Python to PATH" during installation on Windows

Verify Installation: Open your terminal/command prompt and run:

Terminal
python --version # Should show: Python 3.x.x python -m pip --version # Should show: pip version information

2. Set Up VS Code Environment

Install VS Code: Download from code.visualstudio.com

Required Extensions: Install these essential extensions:

  • Python (by Microsoft) - Core Python support
  • Python Test Explorer - For running tests easily
  • Python Docstring Generator - For documentation
  • GitLens - For version control (optional but helpful)

3. Create Your Project Structure

Create a new folder: Create a folder called learn-python-tdd on your computer

Open in VS Code: Open this folder in VS Code

Start with an empty folder: You'll create the necessary files and folders as you work through each chapter

4. Install Required Packages

Create requirements.txt: In your project root, create a file called requirements.txt:

requirements.txt
pytest>=7.0.0 pytest-cov>=4.0.0

Install packages: Run this command in your terminal (from your project folder):

Terminal
pip install -r requirements.txt

5. Configure VS Code Settings

Create .vscode/settings.json: Create a .vscode folder in your project and add a settings.json file:

.vscode/settings.json
{ "python.defaultInterpreterPath": "python", "python.testing.pytestEnabled": true, "python.testing.unittestEnabled": false, "python.testing.pytestArgs": [ "." ], "python.linting.enabled": true, "python.linting.pylintEnabled": false, "python.linting.flake8Enabled": true, "files.autoSave": "afterDelay", "files.autoSaveDelay": 1000 }

6. Test Your Setup

Create a test file: In your project root, create a file called test.py:

test.py
def test_setup(): assert True

Run the test: In VS Code terminal, run:

Terminal
pytest test.py -v

Expected result: The test should pass, confirming that your Python and pytest setup is working correctly!

Getting Started

Once you've completed the setup above, you're ready to begin! Click on "Hello, world" in the sidebar to start your first Python program with TDD!
