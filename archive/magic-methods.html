magic_methods.py
import math class Vector: def __init__(self, x, y): self.x = x self.y = y def __str__(self): return f"Vector({self.x}, {self.y})" def __repr__(self): return f"Vector({self.x}, {self.y})" def __add__(self, other): return Vector(self.x + other.x, self.y + other.y) def __len__(self): return int(math.sqrt(self.x**2 + self.y**2))

What We Learned

  • __init__: Constructor method for object initialization
  • __str__: String representation for user-friendly output
  • __repr__: String representation for debugging
  • __add__: Custom behavior for the + operator
  • __len__: Custom behavior for the len() function
